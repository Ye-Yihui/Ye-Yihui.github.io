使用JS完成简单的数据校验

需求分析

使用JS完成对注册页面的简单数据校验,不允许出现用户名或密码为空的情况

技术分析

html : 它是决定页面的框架

css: 用来美化页面, 将美化内容 和 html 进行分离

js: 提供用户交互

JavaScript概述

什么是javascript:

​ 基于的脚本语言,运行在浏览器中, 由浏览器来解释执行

什么是脚本语言?

​ java — > .class —> jvm运行

​ js – ——> 浏览器直接解释执行

JS的组成:
  • ECMAScript : 定义语法
  • DOM : document Object Model :文档对象模型: 定义了对我们页面结构增删改查的操作
  • BOM : Browser Object Model : 浏览器对象模型: 定义了浏览器的一些行为,前进,后退,刷新, 地址栏,历史记录
JS的语法:
  • 区分大小写
  • 弱类型语言
  • 语句结束之后,它的; 可有可无;
  • 在script标签中去编写js代码
JS的数据类型:
  • 基本类型
    • string
    • number
    • boolean
    • undefine
    • null
  • 引用类型
    • 对象, 内置对象
  • 类型转换
    • js内部自动转换
JS的运算符和语句:
  • 运算符和java 一样
    • “===” 全等号: 值和类型都必须相等
  • 语句和java 一样
JS的输出
  • alert() 直接弹框
  • document.write() 向页面输出
  • console.log() 向控制台输出
  • innerHTML: 向页面输出
  • 获取页面元素: document.getElementById(“id的名称”);
